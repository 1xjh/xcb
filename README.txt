志汇酒店小程序
1.把zh_jd传到微擎或者微赞addons目录

2.小程序应用里面安装

3.新建小程序 设置你自己的信息 提交即可

4.进入小程序 设置模块信息 商品信息等

5.wxapp下修改siteinfo.js
"uniacid":"3",  
"acid":"3"       这两个3都是小程序ID序列号 在系统管理-微信小程序  把鼠标放到进入小程序上，浏览器左下角即可看到



"siteroot":"https://www.weizan.con/app/index.php"   www.we7.cc修改这个为你自己的域名  

然后打开开发者工具  自己调整效果就可以了！
